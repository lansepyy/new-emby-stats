# 字体文件

将字体文件放在这个目录下：

- `title.ttf` - 标题字体（推荐中文字体，如思源黑体、阿里巴巴普惠体等）
- `subtitle.ttf` - 副标题字体（可选，不设置则使用默认字体）

## 推荐字体

### 免费商用字体

1. **思源黑体 (Source Han Sans)**
   - 下载：https://github.com/adobe-fonts/source-han-sans/releases

2. **阿里巴巴普惠体**
   - 下载：https://www.alibabafonts.com/

3. **站酷系列字体**
   - 站酷高端黑：https://www.zcool.com.cn/special/zcoolfonts/

4. **文泉驿字体**
   - 文泉驿微米黑：http://wenq.org/

## 字体安装

将下载的 `.ttf` 或 `.otf` 字体文件重命名为 `title.ttf` 和 `subtitle.ttf`，然后放到这个目录下。

如果不安装字体，程序会使用默认字体，但可能无法正确显示中文。
